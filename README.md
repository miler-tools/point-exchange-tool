## 🗂 収録ツール

### 1. マイル集約電卓・マイル交換レート一覧

**[→ ツールを開く](https://miler-tools.github.io/point-exchange-tool/)**  
ファイル: `index.html`

**目的**：カードは決まっていて、手持ちポイントを集計して目標マイルまでの進捗を確認したい方。

| 機能 | 内容 |
|---|---|
| マイル集約電卓 | モッピー・ハピタス・楽天・Vポイント・MB他を一括入力→ANA/JAL/KF目標への換算合計・達成率を即計算 |
| 交換レートマトリクス | ポイントサイト・共通ポイント・ホテルポイント間の直接・間接ルートを色分け表示 |
| 不足分の現金購入参考額 | マリオット購入経由の補填コストを通常・セール時の3パターンで試算 |
| レート履歴グラフ | 各ルートの改定履歴を可視化 |

### 2. マイル最強クレカ診断ツール（v5）

**[→ ツールを開く](https://miler-tools.github.io/point-exchange-tool/mile-card-tool-v5.html)**  
ファイル: `mile-card-tool-v5.html`

**目的**：年間200〜300万円の決済があり、これからどのクレジットカードで決済すべきかを診断したい方。

#### 主な機能

| 機能 | 内容 |
|---|---|
| 17枚の横断比較 | 初〜中級陸マイラー向け主要カードを収録。広告・提携なしの中立選定 |
| ブランドフィルター | AMEX/JCB/VISA/MC/Dinersを複数選択して絞り込み |
| 年間マイル数リアルタイム試算 | スライダー（100〜400万円）で決済額を変えると全カードの獲得マイル数が即更新 |
| 実質おトク額の自動計算 | 獲得マイル×5円（ビジネスクラス換算）－年会費＝純粋な利得額 |
| 3人家族シンガポールチェッカー | 東京-SIN ビジネスクラス3名分の到達年数をプログレスバーで表示 |
| おすすめ順スコア説明 | 採点内訳を常時表示（透明性確保） |
| 将来の改定見通し | 改悪リスク（低・中・高）と今後の予測を編集部コメント付きで掲載 |
| 紹介経由申込の特典情報 | 各カードの紹介プログラム情報と過去実績を掲載 |
| 公式サイト検索ボタン | 各カードの公式サイトをGoogleで直接検索 |

#### 選定基準

- **対象ユーザー**: 年間200〜300万円の汎用決済を行う日本在住の初〜中級陸マイラー
- **目標マイル**: ANA・JAL・KrisFlyer・マリオットBonvoy・ヒルトンHonorsのいずれか
- **年会費範囲**: 5,500円〜82,500円（超高額プレミアムカードは対象外）
- **還元率基準**: 汎用決済での実質マイル還元率0.3%以上、または特別な選択理由があるカード

