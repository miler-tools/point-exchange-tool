## 🗂 収録ツール

### 1. マイル集約電卓・マイル交換レート一覧

**[→ ツールを開く](https://miler-tools.github.io/point-exchange-tool/)**  
ファイル: `index.html`

**こんな方に**：カードはもう決まっていて、手持ちのポイントを集計して目標マイルまでの進捗を確認したい方。

| 機能 | 内容 |
|---|---|
| マイル集約電卓 | モッピー・ハピタス・楽天・Vポイント・MB他を一括入力→ANA/JAL/KF目標への換算合計・達成率を即計算 |
| 交換レートマトリクス | ポイントサイト・共通ポイント・ホテルポイント間の直接・間接ルートを色分け表示 |
| 不足分の現金購入参考額 | マリオット購入経由の補填コストを通常・セール30%増・50%増の3パターンで試算 |
| レート履歴グラフ | 各ルートの改定履歴を可視化 |
| 改悪アラート | Vポイント・PayPay等の制限事項・2025〜2026年改定を注記 |

**対応ポイント/マイル種別**

- ポイントサイト: モッピー、ハピタス、ファンくる、ドットマネー 他
- 共通ポイント: 楽天、Vポイント（旧T）、PayPayポイント、dポイント、nanacoポイント 他
- ホテルポイント: マリオットBonvoy、ヒルトンHonors 他
- 航空マイル: ANAマイル、JALマイル、KrisFlyer（SQ）、Avios、デルタSkyMiles 他

### 2. マイル最強クレカ診断ツール

**[→ ツールを開く](https://miler-tools.github.io/point-exchange-tool/mile-card-tool-v4.html)**  
ファイル: `mile-card-tool-v4.html`

**こんな方に**：年間200〜300万円の決済があり、これからどのクレジットカードで決済すべきかを診断したい方。

| 機能 | 内容 |
|---|---|
| ブランドフィルター | AMEX/JCB/VISA/MC/Dinersを複数選択して絞り込み |
| 年間マイル数リアルタイム試算 | スライダー（100〜400万円）で決済額を変えると全カードの獲得マイル数が即更新 |
| 実質おトク額の自動計算 | 獲得マイル×5円（ビジネスクラス換算）－年会費＝純粋な利得額 |
| 3人家族シンガポールチェッカー | 東京-SIN ビジネスクラス3名分の必要マイルまでの到達年数をプログレスバーで表示 |
| おすすめ順スコア説明 | スコアの内訳（還元率・こだわり条件マッチ）を常時表示 |
| 将来の改定見通し | 改悪リスク（低・中・高）と今後2〜3年の予測を編集部コメント付きで掲載 |
| 紹介経由申込の特典情報 | AMEXのRefer a Friend、JAL友人紹介等の過去実績と探し方を掲載 |
| 公式サイト検索ボタン | 各カードの公式サイトをGoogleで直接検索できるボタンを配置 |

## 📌 2ツールの使い分け

```
カード選びに迷っている
        ↓
  クレカ診断ツール（mile-card-tool-v4.html）

カードが決まっていて手持ちポイントを集計したい
        ↓
  マイル集約電卓（index.html）
```
